# CA Inter Tracker

A comprehensive offline-first study tracker for CA Intermediate students, built as a single HTML file with optional cloud sync.

## Features

- Fully offline — works from a single HTML file
- Tracks syllabus progress, study sessions, mock tests, revisions
- Optional sync so data survives device changes / browser data clears
- Free and self-owned — no central server
